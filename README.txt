此專案為使用Selenium自動化對官網"網紅配方"進行網路爬蟲
其中皆有兩行是要自行輸入登入帳號密碼
執行順序：先執行"KOL_Index_csv_main.ipynb"獲取內頁連結，再執行"KOL_Info_csv.py"獲取內頁資訊

This project involves using Selenium to automate web scraping on the official website "網紅配方".
There are two lines in the code where you need to input your login credentials manually. 
The execution sequence is as follows: first, run "KOL_Index_csv_main.ipynb" to obtain individual page links, then run "KOL_Info_csv.py" to retrieve information from those individual pages.
